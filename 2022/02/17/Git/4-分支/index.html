创建与合并分支

在该分支上做文件修改并 commit 后，切换回 master 分支

切换回 master 分支后，刚才修改的内容不见了！

因为那个提交是在 newbranch 分支上，而 master 分支此刻的提交点并没有变

这时就需要合并分支

使用 git merge 把 newbranch 的作业成果合并到当前（master）分支上

合并后删除（delete）分支
